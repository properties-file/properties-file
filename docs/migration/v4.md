# Migrating to v4

> This guide covers migrating from v3 to v4. Each migration guide is designed to upgrade from the immediately preceding major version.

## Overview

Version 4 adds bundler integrations for Rollup/Vite, esbuild, and Bun. The only breaking change is the Webpack loader export path.

## Breaking change

### Webpack loader path

The Webpack loader export path changed from `properties-file/webpack-loader` to `properties-file/bundler/webpack`.

_v3_

```js
// webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.properties$/i,
        use: 'properties-file/webpack-loader',
      },
    ],
  },
}
```

_v4_

```js
// webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.properties$/i,
        use: 'properties-file/bundler/webpack',
      },
    ],
  },
}
```

## New features

- Rollup/Vite plugin: `properties-file/bundler/rollup`
- esbuild plugin: `properties-file/bundler/esbuild`
- Bun plugin: `properties-file/bundler/bun`

See [BUNDLER.md](../BUNDLER.md) for setup instructions.
